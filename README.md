# ThumbmarkJS Framework Integrations

Framework-specific integrations for [ThumbmarkJS](https://github.com/thumbmarkjs/thumbmarkjs), the world's best **free** browser fingerprinting JavaScript library.

## Available Packages

- [@thumbmarkjs/react](./packages/react/README.md) - React integration
- [@thumbmarkjs/vue](./packages/vue/README.md) - Vue 3 integration

The integration packages are plugins that require the core [@thumbmarkjs/thumbmarkjs](https://github.com/thumbmarkjs/thumbmarkjs) library.
